# Santos Lisboa 2026

An unofficial interactive platform to discover events, *arraiais* (street parties), and parades for the **Festas de Lisboa 2026**.

## 🌐 Live Access
You can access the platform online here: [**Santos Lisboa**](https://sergiofalves.github.io/santoslisboa/)

## 🛠 About the Project

This project was developed to simplify the search for schedules and locations of the various festivities taking place throughout June in Lisbon. It allows users to search by artist or location, and visualize available options via an interactive map.

- **Developed by:** [Sérgio Alves](https://sergioalves.eu/)
- **Data Source:** The event data was kindly compiled from public sources by [Bruno Braga (Google Sheets)](https://docs.google.com/spreadsheets/d/1pIPzAs4YmoHYlJvc_9-eivOI3GDvnbZT/edit?gid=421994152#gid=421994152).

> **Disclaimer:** This is an unofficial application. The information provided is compiled from public sources and may contain inaccuracies, omissions, or last-minute changes.

## 🚀 Running Locally

To run this project on your local machine:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
